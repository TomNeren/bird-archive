# 🐦 Bird Archive — Vogelstimmen-Quiz

Eine native iOS/macOS-App zum Erkennen heimischer Vogelstimmen.

## Konzept

Höre den Gesang → Erkenne den Vogel → Sammle Wissen.

Jede korrekt erkannte Art wird als **Sammelkarte** ins persönliche Archiv aufgenommen.
Die Karten haben rotierende Fakten — über mehrere Begegnungen lernt man den ganzen Vogel kennen.

## Tech Stack

- **SwiftUI** (iOS 18+ / macOS 15+)
- **AVFoundation** für Audio
- **SwiftData** für Spielstand & Archiv
- **RealityKit** (Phase 2: 3D-Modelle)

## Phase 1: 10 heimische Arten

Amsel · Kohlmeise · Blaumeise · Rotkehlchen · Buchfink · Zaunkönig · Singdrossel · Star · Haussperling · Grünfink
